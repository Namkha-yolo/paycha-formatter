What this application does

Uploads your PDF file safely within your browser

Combines page n with page n+2 (e.g., page 1 with page 3, page 2 with page 4)

Rotates every even-numbered page by 180 degrees after combining

Generates a new PDF document for download

All processing happens locally - your files remain private
